Build Stripe Subscriptions into your WordPress site in minutes. Pricing Pages, checkout and subscription management.

== Description ==

Easily integrate Stripe subscriptions with WordPress using the PriceWell plugin.

== Getting Started ==
 * Install this plugin
 * Create a PriceWell account at [pricewell.io](https://www.pricewell.io/?utm_source=wordpress-plugin) and connect your Stripe account
 * Create a Pricing Page or Customer Portal in PriceWell
 * Insert a shortcode into your WordPress page to embed a Pricing Page or Customer Portal

== Shortcodes ==

= Embed a Pricing Page =
`[pricewell_pricing_page page_id="YOUR_PAGE_ID_HERE"]`

= Embed a Customer Portal =
`[pricewell_customer_portal portal_id="YOUR_PORTAL_ID_HERE"]`

== Installation ==
STEP-BY-STEP INSTRUCTIONS
Log in to your WordPress dashboard (e.g. yourdomain.com/wp-admin).
Click on the "Plugins" tab in the left-hand side, then click "Add New".
Search for "PriceWell" and find the plugin.
Install it by clicking the "Install Now" link.
When installation finishes, click "Activate Plugin".

Further documentation is available [here](https://www.pricewell.io/wordpress-plugin)
